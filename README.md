# rfbridge
Remote Control Bridge using ESP8266 &amp; ESP32. 

## Parts

ESP8266
- 1x ESP8266 ESP12-F
- 1x RF Transmitter 433Mhz
- 1x RF Receiver 433Mhz

ESP32
- 1x ESP32-WROOM32
- 1x RF Transmitter 433Mhz
- 1x RF Receiver 433Mhz
